# GC v1.1.b 

Gestor de Condominio realizado no projeto final de licenciatura em Engenharia Informática por Luís Sousa, 8090228@estg.ipp.pt.

# Tecnologias Usadas

Java, JavaFX, SQLite, JasperReports

# Principais Funcionalidades

Calcular as mensalidades para cada fração num determinado orçamento ou proposta. <br />
Gerar avisos de débito permitindo envio por email ou  impressão. <br />
Gerar recibos permitindo envio por email ou impressão.<br />
Mapa despesas para um determinado orçamento.<br />
Criar relatórios finais para cada orçamento. <br />
